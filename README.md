# Backend Nexus_Hub-app

# Usage

### Clone Repo

```sh
git clone -b backend https://github.com/muhammadisa226/be-tugas-akhir.git
```

### Project Setup

```sh
npm install
```

### Configure database

```sh
npx prisma init
```

```sh
npx prisma migrate dev
```

### Run Server

```sh
npm run dev
```
