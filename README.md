# Backend Nexus_Hub-app

# Usage

### Clone Repo

```sh
git clone -b backend https://github.com/faisalalm01/Nexus_Hub-app.git
```

### Project Setup

```sh
npm install
```

### Configure database

```sh
npm run initdb
```

```sh
npm run generate
```

```sh
npm run migrate
```

### Run Server

```sh
npm run dev
```
